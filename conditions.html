COMMUNIQUÉ

Dans le but de simplifier le processus de vente pour les services confiés aux douanes, plusieurs options sont proposées.

Le souscripteur a la possibilité de s'inscrire plusieurs fois, sachant que chaque inscription correspond à un véhicule aléatoire qui sera proposé au prix de base.

Pour garantir l'obtention du véhicule souhaité, le souscripteur doit s'acquitter d'un acompte de 30% du prix de base lors de son inscription. Le véhicule est alors acquis à ce prix fixe, sans aucune augmentation ni enchère supplémentaire.

Après paiement, le souscripteur sera enregistré dans notre base de données et recevra un code QR associé à son véhicule désiré.

Les 70% restants du prix seront à régler le jour de la vente pour acquérir le nombre de véhicules souhaité.

Conditions Générales

• Les véhicules sont vendus "en l'état". Un rapport de condition est fourni pour inspection.

• Le paiement intégral doit être effectué dans les 48 heures suivant la confirmation.

• Le véhicule doit être retiré sous 5 jours ouvrables après le paiement complet.

• Tous les véhicules présentés sont dédouanés. L'acheteur recevra tous les documents nécessaires pour l'immatriculation.

• Toute inscription est définitive. En cas de non-paiement, l'acompte sera conservé à titre de pénalité.
